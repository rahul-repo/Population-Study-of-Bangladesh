# Population Study of Bangladesh (2025)

## Overview

This project presents a demographic analysis of Bangladesh using census and population data from 1951–2023. The study investigates long-term population structure, demographic transition patterns, and changes in dependency dynamics through statistical and graphical analysis.

Special emphasis is placed on population pyramids, working-age population growth, and dependency ratios to understand demographic changes over time.

---

## Objectives

- Analyze long-term population trends in Bangladesh
- Study demographic transition patterns across decades
- Construct and interpret population pyramids
- Examine changes in dependency structure
- Compare demographic indicators with global trends

---

## Data

The analysis uses historical demographic and census datasets covering:

- Population counts (1951–2023)
- Age-group distributions
- Gender-wise population statistics
- Working-age population estimates
- Global demographic comparison indicators

---

## Methodology

### Population Structure Analysis

- Construction of population pyramids
- Age-group distribution analysis
- Gender composition assessment

### Demographic Transition Analysis

- Study of population growth patterns
- Identification of demographic transition stages
- Analysis of structural population change over time

### Dependency Analysis

- Child Dependency Ratio (CDR)
- Adult Dependency Ratio (ADR)
- Total Dependency Ratio (TDR)
- Working-age versus dependent population comparison

### Comparative Analysis

- Bangladesh demographic trends
- Comparison with global demographic patterns
- Assessment of population ageing and dependency changes

---

## Key Contributions

- Analyzed more than seven decades of demographic data
- Constructed and interpreted population pyramids
- Studied demographic transition dynamics
- Evaluated dependency structure using CDR, ADR, and TDR
- Compared Bangladesh's demographic evolution with global trends
